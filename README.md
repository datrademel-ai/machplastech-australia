# Machplastech Australia

Korea-Australia Industrial Gateway

## About

Machplastech Australia connects Australian businesses with Korean industrial technology, providing comprehensive trade facilitation services.

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Static Export

## Installation
```bash
npm install
```

## Development
```bash
npm run dev
```

Visit http://localhost:3001

## Build
```bash
npm run build
```

## Deploy

Static site - deploy the `out` folder to any hosting service.
